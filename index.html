第1期-單獨會主

Posted on 2018-01-19 | In 2018年單獨會主

日期:2018-1-19
經文: 加 3: 11-20

1. 關於上帝

基督為我們的罪受了咒詛、守約、耶穌基督是我們的中保

2. 關於我

脫離了律法的咒詛、得着聖靈、

3. 關於信徒的生活(SPECT)

  • Sin
    不信、想靠行為稱義

  • Example

  • Promises

  • Command
    要信神

  • Truth
    律法原不在乎信

4. 回應

  • 對不起
    在工作上不忠心。公司領導叫我去debug，但覺得找bug很麻煩，所以每次都說找不到。其實压根没用心找。在這個公司就只有我做軟件，軟件部就我一人，平時上班做的事其他人都不懂，所以没有人考核和監管我。早在上班兩天後就發現這公司没有會編程的，所以他們不知道我平均需要多長的時間去解決問題。上次一個我40分鐘就能解決的問題，硬是給我拖了三天。有時候其實早把問題解決了，卻向上報告。一來為了防止有人給我佈置新任務，二來利用這段時間差去為自己學新知識，二次開發公司項目成為自己的項目，偷取上班時間。
  • 態度/行動
    無論是在大事或小事上都要忠心，不能因為別人不懂技術就去忽悠人。花時間去研究代碼，把問題修復，在這事情上操練忠心。
  • 祈求
    求主給我忠心，也給我屬天的智慧。
  • 謝謝

第7期-單獨會主

Posted on 2018-01-18 | In 2018年單獨會主

日期:2018-1-18
經文: 加 3: 1-10

1. 關於上帝

為我們被釘十字架、內住在人裡面

2. 關於我

亞伯拉罕的子孫、被神稱義

3. 關於信徒的生活(SPECT)

  • Sin
    以行律法為本
  • Example
    亞伯拉罕，以信為本。對神的話語應許有信心。
  • Promises
    以信為本的人，就是亞伯拉罕的子孫
    以信為本的人可以和亞伯拉罕一同得福
    義人必因信得生
  • Command
    信靠神
  • Truth
    没有人可以靠着行律法在神的面前稱義

4. 回應

  • 對不起
    爸爸今天去做手術，没有去主動關心，也不知道原來他今天去做手術。没有為這擺上禱告。

  • 態度/行動
    打電話給爸爸去關心他，為他禱告。

  • 祈求
    為爸爸的身體禱告，在這事情上更多經歷神。

  • 謝謝
    感謝神我們弟兄四人在晚上有彼此禱告交通的時間。

第6期-單獨會主

Posted on 2018-01-17 | In 2018年單獨會主

日期:2018-1-17
經文: 加 2: 11-21

1. 關於上帝

愛我的神，祂為我的罪捨己。在人裡面活着的。

2. 關於我

蒙神所愛的人。因信基督而稱義。

3. 關於信徒的生活(SPECT)

  • Sin
    怕得罪人而行與真理不乎的事。信主之後生命没有改變，回來以前老我的樣子。想靠着行律法稱義。

  • Example
    保羅見彼得有做的與福音真理不合的地方，直接提出來。保羅為神而活，他所做的一切都是為了神。

  • Promises
    因信稱義

  • Command
    信基督耶穌、向神活着、為神而活。
  • Truth
    人稱義不是因行律法，及是因信耶穌基督。
    凡有血氣的，没有一人因行律法稱義。

4. 回應

  • 對不起
    没有為神而活，現在每天都是為自己而活。不去思想耶穌為我捨己的愛，對祂的愛没有回應。

  • 態度/行動
    每天思想神的愛，每天都記下神對自己的恩典，記下感恩的事項。為主而活，在跟神的關係上多付上代價、時間。

  • 祈求
    求主讓我更多明白主對我的愛有多麼的長闊高深。使我成了一個為主而活的人。求主保守改珍爸爸下星期的手術，讓他在這當中更多的去經歷神。

  • 謝謝
    感謝主帶領我這一天。

第5期-單獨會主

Posted on 2018-01-16 | In 2018年單獨會主

日期:2018-1-16
經文: 加 2: 1-10

1. 關於上帝

不以外貌取人、賜人恩典

2. 關於我

傳福音的，為主做工

3. 關於信徒的生活(SPECT)

  • Sin
    假弟兄、在乎別人的肯定
  • Example
    保羅唯一的渴慕就是討好的喜悅。他不去討人的喜悅，也不在乎別人的眼光。保羅對主所發的熱心是持續的，不是間間斷斷的。
  • Command
    紀念窮人，傳福音

4. 回應

  • 對不起
    把時間和心思都放在工作上及學習上，在這個期間完全把神把在一旁。想利用工作為自己得到滿足感。自己不知不覺的把工作和自己的私慾當成偶像。
    消滅聖靈的感動，没有去關心弟兄姐妹。

  • 態度/行動
    把晚上的時間分別出來用來親近神。抽到時間去關心肢體們，與他們交通。

  • 祈求
    求神把我心中的偶像打碎，使我愛主更深。求主親自去吸引我的同事，把渴慕神的心放在他們裡面，也求主使用我成為福音的管道。

  • 謝謝
    今天班上的同事主動問我有關福音、信仰的事情。感謝祢，藉着他們來提醒我，我是屬祢的，而非屬世界的。我現在的狀況，我所經歷的祢都知道。感謝祢一直的在暗中看顧着我，引導我。

日期:2018-1-8
經文: 加 1:11-24

1. 關於上帝

2. 關於我

3. 關於信徒的生活(SPECT)

  • Sin

  • Example

  • Command

  • Truth

4. 回應

  • 對不起
    因為上週內突然要去出差，所以心裡煩躁。當自己心裡煩躁時，就給自己找理由不去做單獨會主。在與神的關係中，仍舊是以自己為中心。

  • 謝謝
    感謝主，把我所不能承受的重擔拿走。我所經歷的，祢都知道，並在暗中幫助我。

第3期-單獨會主

Posted on 2018-01-03 | In 2018年單獨會主

日期:2018-1-3
經文: 加 1:6-10

1. 關於上帝

祂是不變的、永恆的

2. 關於我

基督的僕人

3. 關於信徒的生活(SPECT)

  • Sin
    去跟從別的福音，對真理没有辨別的能力。有人把基督的福音更改，並且去傳給別人。討人的喜悅，得人的心。
  • Example
    保羅所做的一切事，都是為了討神的喜悅，得神的心。
  • Command
    要得神的心
  • Truth
    傳假福音的人應當被咒詛
    不能同時去討人喜悅和去討神的喜悅

4. 回應

  • 對不起
    會為了自己的利益而去討人的喜悅。知道有時候做討神喜悅的事，會使別人的利益受損，從而自己的利益也會受損。所以有時會不去做討神喜悅的事。不知道自己活着最大的目的是為了討神的喜悅。
  • 態度/行動
    讓自己明白，我一切的好處都不在神以外。讓自己專心去做討神喜悅的事，放下自己。
  • 祈求
    讓我更明白神的心意。讓我更多的放下自己，做祢所喜悅的事。
  • 謝謝
    感謝主藉着聖靈的感動，使對祢有更多的渴慕。

第2期-單獨會主

Posted on 2018-01-02 | In 2018年單獨會主

日期:2018-1-2
經文: 加 1:1-5

1. 關於上帝

耶穌基督為我的罪受死，並且復活。恩惠和平安是從神而來的。

2. 關於我

神僕人的權柄是來自於神，而不是來自於人。神救我脫離罪惡的世代。

3. 關於信徒的生活(SPECT)

  • Sin
    不順服神、没有捨己、没有歸榮耀給神。
  • Promises
    給我們恩惠與平安、救我們脫離這個罪惡的世代。
  • Example
    耶穌照父神的旨意，去為罪人捨己。
  • Command
    榮耀神
  • Truth
    基督為我們捨命，來救贖我們。

4. 回應

  • 對不起
    求主赦免我的不順服，經常消滅來自聖靈的感動。選擇性地去順服，而不是像主耶穌那樣無條件的順服神每一個的帶領。
  • 態度/行動
    對聖靈有一個敬畏的心，專心去聆聽祂的聲音。把聖靈感動我去做的事情記錄下來，並且去行，以此去操練順服的功課。

  • 祈求
    求主給我順服、捨己的心。

  • 謝謝
    感謝主給我預備時間，讓我可以去靈修讀經。

第1期-單獨會主

Posted on 2018-01-01 | In 2018年單獨會主

1. 關於上帝

知道萬人的心、親自去牧養屬祂的百姓、有屬天的權柄

2. 關於我

神的百姓、接受神教訓的人

3. 關於信徒的生活(SPECT)

  • Sin
    不明白真理、也不去領受或思想從神而來的教訓。終日所想的都是自己在世上的利益，没有思想屬天的事。
  • Example
    挪亞，他對神有一顆敬畏的心。他因着對神的信心，做一隻方舟。他的生活以神為中心。他所做的事，都是為了討神的喜悅。

4. 回應

  • 對不起
    在日常生活中，没有想着去討神的喜悅。自己所定下來的目標都是為了討自己的喜悅。没有去尋求神的帶領。去年，完全跟着自己的意思走。上一年自己所定的目標神都應允了，但現在反而變得迷失，覺得這不是我想要的東西。
  • 態度/行動
    在新的一年向神立志要做一個討神喜悅的人。每天跟着教會的步伐去讀經和去做單獨會主。
  • 祈求
    求神給我一個目標，告訴我祂想要我去為祂做什麼，或是去預備什麼。
  • 謝謝
    感謝祢天天都看顧我，感謝祢藉着弟兄姐妹來讓我感受到你的愛。
